Following task are coded in Run_Analysis.R  file
•	Retrieve all data sets  i.e train or test  from  local drive
•	Retrieve features.txt and extract only mean and standard deviation measurements indices from it
•	Use these indices to extract out required data variables from measurements of  test 
•	Merge all separated train and tests data sets into a data frame merge_data
•	Rename all variables with proper understandable variable names in the merged dataset
•	Label activities of subject in dataset with respective given labels
•	Calculate and store mean of every measurements on the basis of activities of a particular subject
•	Write data frame into a file named tidy_data.txt
